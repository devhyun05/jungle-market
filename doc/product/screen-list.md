# 화면 목록 정의서

## 1. 문서 목적

이 문서는 크래프톤 정글 내부 중고거래 웹서비스 MVP에 필요한 화면 목록과 각 화면의 목적, 주요 기능, 표시 데이터, 이동 경로를 정의한다.

본 문서는 다음 작업의 기준으로 사용한다.

- 와이어프레임 설계
- 프론트엔드 라우팅 설계
- API 명세 작성
- 컴포넌트 분리
- 테스트 시나리오 작성
- GitHub Projects 태스크 분리

## 4. 공통 화면

## 4.1 로그인 화면

| 항목        | 내용                                               |
| ----------- | -------------------------------------------------- |
| 화면 ID     | SCR-001                                            |
| 화면명      | 로그인 화면                                        |
| 화면 목적   | 정글 내부 사용자가 서비스에 로그인할 수 있게 한다. |
| 접근 사용자 | 비로그인 사용자                                    |
| 진입 경로   | 서비스 최초 진입, 보호된 URL 접근                  |
| 이동 화면   | 상품 목록 화면, 접근 불가 화면                     |

### 주요 기능

- 내부 사용자 로그인
- 로그인 실패 메시지 표시
- 비인가 사용자 접근 제한 안내
- 정지된 사용자 접근 제한 안내

### 표시 데이터

- 서비스명
- 로그인 입력 폼
- 로그인 버튼
- 오류 메시지

### 관련 요구사항

- AUTH-001
- AUTH-002
- NFR-SEC-001

| 화면명      | 접근 불가 화면                                                           |
| 화면 목적   | 권한이 없는 사용자가 제한된 화면에 접근했을 때 안내한다.                 |
| 접근 사용자 | 비로그인 사용자, 일반 사용자, 정지된 사용자                              |
| 진입 경로   | 관리자 페이지 접근 실패, 정지 사용자 기능 접근, 권한 없는 수정/삭제 시도 |
| 이동 화면   | 로그인 화면, 상품 목록 화면                                              |

### 주요 기능

- 접근 불가 안내
- 이전 화면으로 이동
- 상품 목록으로 이동
- 로그인 화면으로 이동

### 표시 데이터

- 접근 불가 메시지
- 사유 안내
- 이동 버튼

### 관련 요구사항

- AUTH-002
- AUTH-005
- ADMIN-002
- NFR-SEC-001
- NFR-SEC-003

## 5. 사용자 화면

## 5.1 상품 목록 화면

| 항목        | 내용                                                           |
| ----------- | -------------------------------------------------------------- |
| 화면 ID     | SCR-002                                                        |
| 화면명      | 상품 목록 화면                                                 |
| 화면 목적   | 사용자가 등록된 상품을 검색, 필터링, 탐색할 수 있게 한다.      |
| 접근 사용자 | 로그인 사용자                                                  |
| 진입 경로   | 로그인 성공, 상품 상세에서 목록 이동, 마이페이지에서 목록 이동 |
| 이동 화면   | 상품 상세 화면, 상품 등록 화면, 마이페이지                     |

### 주요 기능

- 상품 목록 조회
- 키워드 검색
- 카테고리 필터
- 판매중 상품만 보기
- 최신순 정렬
- 상품 등록 화면 이동
- 상품 상세 화면 이동
- 마이페이지 이동

### 표시 데이터

| 데이터           | 설명                        |
| ---------------- | --------------------------- |
| 상품 대표 이미지 | 목록에서 보이는 대표 이미지 |
| 상품 제목        | 상품명                      |
| 가격             | 판매 가격                   |
| 카테고리         | 상품 분류                   |
| 거래 장소        | 희망 거래 위치              |
| 판매 상태        | 판매중, 예약중, 거래완료    |
| 등록 시간        | 상품 등록 시점              |

### 주요 액션

| 액션             | 결과                      |
| ---------------- | ------------------------- |
| 검색어 입력      | 검색 결과 목록 표시       |
| 카테고리 선택    | 선택 카테고리 상품만 표시 |
| 판매중 필터 선택 | 판매중 상품만 표시        |
| 상품 카드 클릭   | 상품 상세 화면으로 이동   |
| 상품 등록 클릭   | 상품 등록 화면으로 이동   |
| 마이페이지 클릭  | 마이페이지로 이동         |

### 예외 상태

| 상황           | 처리                           |
| -------------- | ------------------------------ |
| 상품이 없음    | 빈 상태 화면 표시              |
| 검색 결과 없음 | 검색 결과 없음 메시지 표시     |
| 서버 오류      | 오류 메시지와 재시도 버튼 표시 |

## 5.3 상품 등록 화면

| 항목        | 내용                                    |
| ----------- | --------------------------------------- |
| 화면 ID     | SCR-004                                 |
| 화면명      | 상품 등록 화면                          |
| 화면 목적   | 사용자가 새 상품을 등록할 수 있게 한다. |
| 접근 사용자 | 로그인 사용자                           |
| 진입 경로   | 상품 목록 화면에서 상품 등록 클릭       |
| 이동 화면   | 상품 상세 화면, 상품 목록 화면          |

### 주요 기능

- 상품 제목 입력
- 가격 입력
- 설명 입력

- 거래 장소 입력
- 이미지 업로드
- 등록 요청
- 입력값 검증

| 입력값    | 필수 여부 | 설명                              |
| --------- | --------- | --------------------------------- |
| 제목      | 필수      | 상품 제목                         |
| 가격      | 필수      | 판매 가격                         |
| 설명      | 필수      | 상품 설명                         |
| 카테고리  | 필수      | 상품 분류                         |
| 상품 상태 | 필수      | 새상품, 거의 새것, 사용감 있음 등 |
| 거래 장소 | 필수      | 희망 거래 위치                    |
| 이미지    | 필수      | 1개 이상                          |

### 예외 상태

| 상황                      | 처리                         |
| ------------------------- | ---------------------------- |
| 필수 입력값 누락          | 누락된 항목 안내             |
| 이미지 미첨부             | 이미지 등록 필요 메시지 표시 |
| 허용되지 않은 이미지 형식 | 업로드 불가 메시지 표시      |
| 업로드 실패               | 재시도 안내                  |
| 정지된 사용자 접근        | 접근 불가 화면 표시          |

| 데이터         | 설명                                 |
